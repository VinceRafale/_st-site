== Description ==

Simply insert your google meta tag verification code using this helpful plugin.

Once your blog is verified, you can get your search statistics, Malware warning, Index status, HTML suggestion and many more google webmaster features.

If you are using caching plugins like WP-Cache or WP Super Cache, you may need to temporarily clear or disable your cache for this plugin to work.

== Installation ==

1. Upload Google site verififaction plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your verification id in plugin settings section as mentioned in screenshot.
4. Clear your cache if you are using anay cache plugin.
5. All Done.
